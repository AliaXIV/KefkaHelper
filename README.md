# Kefka Helper

Simple plugin assisting with Kefka Ultimate.

As of now the marking is semi-automatic, plugin will check for the mechanic and internally assign marks but the actual marking has to be done by user action (pressing button in plugin window). 

## Features
- Semi automarker for blackhole - automatically recognizes the marking but requires an user action to mark people: one by one, only accretion players or all players. Marking is staggered around 1s per mark with some variance.
- Forsaken Future/Past chat notification
- Forsaken hidden debuff display

## Work in progress
